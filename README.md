# koa-example

```
npm install
npm install -g nodemon
nodemon ./app
```

#### 路由
```
npm install koa-router --save
```

#### 参数解析
```
npm i koa-body --save
```

#### 跨域模块
```
npm install koa2-cors --save
```

#### jwt token生成校验
```
npm install jsonwebtoken --save
```
